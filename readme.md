## 概要
吃音当事者団体の情報を構造化し、地域別検索・可視化・再利用を目的としたデータベースです。

## できること
- 地域別の団体検索
- 活動状況（active/inactive）の把握
- 外部サービスでの再利用（Web・アプリ・研究等）

### 想定利用例
- 吃音当事者向けの団体検索サイト
- 地図ベースの可視化アプリ
- 福祉・アクセシビリティ研究

### 免責事項

- 本データベースの情報は公開情報をもとに個人が収集・整理したものです
- 掲載情報の正確性・最新性は保証しません
- 団体の活動状況・連絡先等は変更される場合があります
- 最新情報は各団体の公式サイト・SNSを必ずご確認ください
- 情報の誤り・更新はIssueまたはPull Requestにてお知らせください

## データについて

- 全言連加盟団体：[全国言友会協議会](https://www.zengenren.org/)の公開情報をもとに作成
- その他団体：各団体の公開情報をもとに作成
- 最終確認日をJSONの`last_confirmed`フィールドに記載
- 本サイトは個人が運営しており、定期更新の保証はありません。

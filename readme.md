## 吃音団体データベースJP（JPST-DB）：概要
吃音当事者団体の情報を構造化し、地域別検索・可視化・再利用を目的としたデータベースです。  
地域別検索や可視化など、Webサービスや研究用途での再利用を想定しています。

## 特徴
- JSON形式で管理
- 活動状況（active / inactive）を明示
- 最終確認日を記録

### データ例
> 一例を紹介します
"id": "org-tyo-001",
"name": "東京言友会",
"region_level1": "関東",
"region_level2": "東京都",
"region_level3": ["東京23区","多摩地域"],"affiliation": "全言連",
"url": "https://tokyo-gennyukai.jimdofree.com/",
"twitter": "https://x.com/tokyo_genyukai2",
"note": "1960年代より活動し、60周年を迎える。メディア協力等の実績あり。月二回の例会を巣鴨と東京障害会館（三田）にて行う。",
"tags": ["成人", "定例会","吃音改善"],
"status": "active", 
"last_confirmed": "2026-04"

### できること
- 地域別の団体検索
- 活動状況（active/inactive）の把握
- 外部サービスでの再利用（Web・アプリ・研究等）

### 想定利用例
- 吃音当事者向けの団体検索サイト
- 地図ベースの可視化アプリ
- 福祉・アクセシビリティ研究（福祉分野のデータ活用）

## 免責事項

- 本データベースの情報は公開情報をもとに個人が収集・整理したものです
- 掲載情報の正確性・最新性は保証しません
- 団体の活動状況・連絡先等は変更される場合があります
- 最新情報は各団体の公式サイト・SNSを必ずご確認ください
- 情報の誤り・更新はIssueまたはPull Requestにてお知らせください

## データについて

- 全言連加盟団体：[全国言友会協議会](https://www.zengenren.org/)の公開情報をもとに作成
- その他団体：各団体の公開情報をもとに作成
- 最終確認日をJSONの`last_confirmed`フィールドに記載
- 本サイトは個人が運営しており、定期更新の保証はありません。
